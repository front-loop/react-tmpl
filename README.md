# React Vite Tailwind Tmpl

Built on the following stack:

- React
- Vite
- Bun
- Typescript
- Tailwind CSS
- React Router
- Antfu's Eslint Config
- Simple-git-hooks + lint-staged
- Taze

## Usage

```sh
npx degit front-loop/react-vite-tailwind-tmpl my-app 
cd my-app
ni # npm i -g @antfu/ni
```
